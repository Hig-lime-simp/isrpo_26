## Основные выводы

В ходе выполнения лабораторной работы были изучены и применены на практике ключевые возможности современного JavaScript:

### 1. Деструктуризация
- Упрощает извлечение данных из массивов и объектов
- Позволяет задавать значения по умолчанию
- Используется в параметрах функций

```javascript
const { name, age } = user;
const [first, second, ...rest] = array;
```
2. Spread и Rest операторы
Spread (...) — разворачивает массивы и объекты

Rest (...) — собирает элементы в массив

```javascript
const combined = [...arr1, ...arr2];
const [first, ...others] = numbers;
```
3. Модули (Import/Export)
Позволяют организовать код в отдельных файлах

Именованные экспорты и default экспорт

Повышают переиспользование и изоляцию кода

```javascript
export class TaskManager { ... }
import TaskManager from './TaskManager.js';
```
4. Промисы (Promises)
Объект, представляющий результат асинхронной операции

Состояния: pending → fulfilled / rejected

Методы:javascript .then(), .catch(), .finally()

5. Async/Await
Современный способ работы с асинхронным кодом

Делает код более читаемым, похожим на синхронный

Использует try/catch для обработки ошибок

```javascript
async function fetchData() {
    try {
        const response = await fetch(url);
        const data = await response.json();
        return data;
    } catch (error) {
        console.error(error);
    }
}
```
6. Fetch API
Современный стандарт для HTTP-запросов

Поддерживает GET, POST, PUT, DELETE методы

Работает с промисами

7. Optional Chaining (?.) и Nullish Coalescing (??)
?. — безопасный доступ к вложенным свойствам

?? — значение по умолчанию только для null/undefined

```javascript
const city = user?.address?.city ?? "Не указан";
```
